搜索功能

在各大搜索引擎顶部展示快捷其他搜索引擎, 快捷切换

1. 聚合搜索引擎切换 推荐

  • 在屏幕最上方有一排引擎供快捷切换,展开后支持快捷搜索其他网站

  • 脚本内修改简单

2. all-search全搜,搜索引擎快捷跳转,支持任意网站展示

  • 脚本是有完整的设置页面

  • 观感舒服,操作华丽,修改复杂

3. 一键切换搜索

  • 在屏幕左侧中心偏上有较小的侧边栏

  • 不影响观感和操作,脚本内修改简单

4. 搜索引擎切换器2(侧栏版)

  • 在屏幕左侧中心有较小的侧边栏,点击可展开

  • 不影响观感和操作,脚本内修改简单


链接功能

将网页的文本转为超链接,又或者是自动跳转重定向...

1. External Link Auto Redirect(Direct Link) --- 外部链接自动重定向

  • 单击包含重定向URL的链接时,直接重定向到真实URL

  • 通用规则,无需自行添加(但有小部分按钮误杀,点击无效果)

2. redirect外链跳转

  • 酷安,贴吧,知乎一系列.的外部链接,自动跳转,规则简单

3. Open the F**king URL Right Now

  • 自动跳转某些网站不希望用户直达的外链

4. anti-redirect --- 反重定向

  • 去除各搜索引擎/常用网站的重定向

5. 链接助手

  • 可对单个网站自定义链接规则在新标签页中打开

  • 文本链接转为超链接...等

6.让网页中的文字链接变为可点击

  • Linkify Plus Plus 推荐

  • Text To link

  • 测试链接是否可直接点击的网址

7. 链接地址洗白白

  • 把链接地址缩减至最短可用状态,并复制到剪切板

  • 在网页底部中间,有一个按钮,用来呼出面板

8. 短链生成

  • 打开脚本菜单,点击生成"短链按钮"即可将当前网址转为短链,并自动复制到剪贴板

翻译脚本

翻译网页内容的脚本，除第4个与第5个,其他均可在Github上运行

1. Immersive-translate --- 沉浸式翻译 商业化,不开源

  • 可“显示原文(双显)”和“隐藏原文(仅显示译文)”

  • 历史版本

2. 网页中英双显互译(其他语言也可以翻译) 推荐,开源

  • 可“显示原文(双显)”和“隐藏原文(仅显示译文)”

3. KISS Translator --- 亲吻翻译

  • 只有“显示原文(双显)”

4. 网页翻译 不可在Gayhub等网站运行

  • 给每个非中文的网页右下角（可以调整到左下角）添加一个 google 翻译图标, 直接调用 Google 的翻译接口对非中文网页进行翻译

  • 需要外网环境

5. YanDex翻译 不可在Gayhub等网站运行

  • 使用 Yandex Translate API 对网页进行翻译
